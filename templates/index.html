Telegram Music Bot

A sophisticated bot for playing Spotify songs in Telegram voice chats

Open Bot in Telegram

Features

  • Spotify Integration - Play music directly from Spotify
  • User Profiles - Personalized settings for each user
  • Queue Management - Add, remove, and reorder songs in the queue
  • Favorites - Save your favorite songs for quick access
  • Interactive UI - Control playback with inline buttons
  • Statistics - Track most played songs and user activity
  • Multi-language Support - Use the bot in your preferred language

Commands

Basic Commands

/start or /help

Start the bot and see the available commands

/play [song name or Spotify URL]

Play a song or add it to the queue

Example: /play Billie Jean or /play https://open.spotify.com/track/5ChkMS8OtdzJeqyybCc9R5

/search [query]

Search for a song on Spotify

Example: /search Shape of You

/pause

Pause the current playback

/resume

Resume the paused playback

/skip

Skip to the next song in the queue

/stop

Stop playback and leave the voice chat

/queue

Show the current song queue

/current

Show information about the currently playing song

User Commands

/profile

View your user profile with statistics and preferences

/settings

Configure your personal preferences

/favorite

Add the currently playing song to your favorites

/favorites

List your favorite songs

Advanced Commands

/volume [0-200]

Set the playback volume (default: 100)

Example: /volume 75

/ping

Check the bot's response time

/stats

Show statistics about the bot usage in the current chat

How to Use

  1. Add the bot to a Telegram group
  2. Join a voice chat in the group
  3. Use /play to start playing music
  4. Control playback with commands or inline buttons
  5. Customize your experience with /settings
